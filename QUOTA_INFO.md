# OpenAI API クォータ情報

## 現在の状況

APIキーは正しく設定されており、認証も成功していますが、クォータ超過エラー（429）が発生しています。

## 対処方法

1. **OpenAI Platformでクレジットを確認**
   - https://platform.openai.com/account/billing にアクセス
   - クレジット残高を確認

2. **クレジットを追加**
   - 必要に応じてクレジットを追加してください

3. **無料プランの場合**
   - 無料プランには月間の使用制限があります
   - 制限内で使用するか、有料プランにアップグレードしてください

## 実行コマンド

クォータが回復したら：

```bash
# 疎通確認
make smoke

# ウォームアップ実行
make warmup

# 本番ベンチマーク
make profile

# サマリ生成
make summary
```

## モデル情報

現在のモデル: `gpt-4o-mini`
- より小さいモデル: `gpt-3.5-turbo`（より安価）
- より大きいモデル: `gpt-4o`（より高価）
